Thanks for using the Resource Locker! Here are the latest feature updates:

In the Lockable Resources page, in the table, now each resource name is also a clickable link that will display more metadata about the resource. This metadata page displays the time that the resource has been locked, also calculates the total time and displays the total amount in a descriptive text. The locked period will be displayed under the red colored LOCKED sign in a shorter description as well.
In the Dashboard, in the section Available Resource and Unavailable Resources, inside the secondary section of unavailable resources, the table of locked or in maintenance mode will be SORTED by locked period. This means, that the older the resource is being locked (used), the higher it will be located in this table. If a resource is unavailable because it is in maintenance mode, it is likely to be located in the bottom section of this table (since it's probably not going to have a locked period). The Available resources however, will remain to be sorted by their ID (ascending). In addition to mentioned above, the table will also display the total time that the resource is locked.
Happy Resource Locking!
